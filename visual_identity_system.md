## üé® Design Principles

### Visual Hierarchy
1. **Primary**: Logo, main headline, key CTA
2. **Secondary**: Section headers, important metrics
3. **Tertiary**: Body text, descriptions, details

### Consistency Rules
- **Spacing**: Use 8px grid system
- **Corners**: 8px border radius for cards
- **Shadows**: Subtle depth with consistent shadow
- **Contrast**: Minimum 4.5:1 for accessibility

### Professional Elements
```css
/* Cards & Components */
```
