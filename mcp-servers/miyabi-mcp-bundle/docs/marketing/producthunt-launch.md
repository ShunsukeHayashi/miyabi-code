## Description (Long)
Miyabi MCP Bundle is an all-in-one MCP (Model Context Protocol) server that supercharges your Claude Desktop with 102 developer tools.

**What you can do:**
- Git operations: commit history, diff, blame, branch management
- GitHub integration: create issues, manage PRs, monitor workflows
- System monitoring: CPU, memory, disk usage tracking
- File operations: watch, search, backup
- And 80+ more tools

**Why it's different:**
- One server, 102 tools (vs 5-20 tools in typical MCP servers)
- Tools work together seamlessly
- Japanese + English documentation
- 1-minute installation

**Perfect for:**
- Developers who live in Claude Desktop
- Teams wanting to automate Git/GitHub workflows
- Anyone tired of context switching between tools

## Maker Comment
"I built Miyabi MCP Bundle because I was frustrated with managing multiple MCP servers. Now everything I need is in one place - I just ask Claude and it handles the rest. From git log to creating GitHub issues, it's all natural language."
