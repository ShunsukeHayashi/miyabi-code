# Google Apps Script - Slides自動生成

このディレクトリには、AI駆動開発カンファレンス 2025秋のプレゼンテーション資料を自動生成するGoogle Apps Script (GAS) が含まれています。

## 📋 ファイル一覧

- `create-slides.gs` - Google Slidesを自動生成するGASスクリプト

## 🚀 使用方法

### ステップ1: Google Apps Scriptエディタを開く

1. Google Driveにアクセス: https://drive.google.com
2. 「新規」→「その他」→「Google Apps Script」をクリック
3. 新しいプロジェクトが作成されます

### ステップ2: スクリプトをコピー

1. `create-slides.gs` の内容をコピー
2. Google Apps Scriptエディタに貼り付け
3. プロジェクト名を「Miyabi Slides Generator」に変更（任意）
4. 保存（Ctrl+S または ⌘+S）

### ステップ3: スクリプトを実行

#### 方法A: 全40スライドを作成

1. 関数選択: `createPresentation`
2. 「実行」ボタンをクリック
3. 初回実行時: 権限の承認が必要
   - 「権限を確認」をクリック
   - Googleアカウントを選択
   - 「詳細」→「Miyabi Slides Generator（安全ではないページ）に移動」をクリック
   - 「許可」をクリック
4. 実行完了後、ログにプレゼンテーションURLが表示されます

#### 方法B: テスト（最初の3スライドのみ）

1. 関数選択: `testCreatePresentation`
2. 「実行」ボタンをクリック
3. ログにテストプレゼンテーションURLが表示されます

#### 方法C: 特定範囲のスライドを作成

1. 関数選択: `createSlidesRange`
2. スクリプト内でパラメータを編集:
   ```javascript
   // 例: スライド1-10を作成
   function runCreateSlidesRange() {
     return createSlidesRange(0, 9);  // 0-indexed
   }
   ```
3. 新しい関数 `runCreateSlidesRange` を実行

### ステップ4: プレゼンテーションを確認

1. ログに表示されたURLをクリック
2. Google Slidesでプレゼンテーションが開きます
3. 内容を確認・編集

## 🎨 デザインカスタマイズ

### カラーパレットの変更

`create-slides.gs` の `PRESENTATION_CONFIG` を編集:

```javascript
colors: {
  primary: '#2563eb',   // 青 - Miyabiメインカラー
  secondary: '#10b981', // 緑 - 成功・完了
  accent: '#f59e0b',    // オレンジ - 警告・注目
  error: '#ef4444',     // 赤 - エラー・失敗
  background: '#ffffff', // 白 - 背景
  text: '#1f2937'       // 濃いグレー - 本文
}
```

### フォントの変更

生成後、Google Slidesの「テーマを編集」で一括変更可能:

1. 「スライド」メニュー → 「テーマを編集」
2. マスタースライドでフォントを変更
3. 全スライドに適用

### レイアウトの変更

`PRESENTATION_CONFIG.layout` を編集:

```javascript
layout: {
  width: 720,  // 10インチ (16:9)
  height: 405,
  marginTop: 40,
  marginBottom: 40,
  marginLeft: 60,
  marginRight: 60
}
```

## 🔧 トラブルシューティング

### エラー: 権限がありません

**原因**: Google Apps Scriptがスライドを作成する権限がない

**解決策**:
1. スクリプト実行時に「権限を確認」をクリック
2. Googleアカウントでログイン
3. 「詳細」→「安全ではないページに移動」をクリック
4. 「許可」をクリック

### エラー: スライドが作成されない

**原因**: スクリプトにエラーがある

**解決策**:
1. 「表示」メニュー → 「ログ」を開く
2. エラーメッセージを確認
3. スクリプトの該当箇所を修正

### スライドの内容が正しくない

**原因**: `getSlidesData()` 関数のデータが不正

**解決策**:
1. `getSlidesData()` 関数内のスライドデータを確認
2. 各スライドの `title`, `content`, `speakerNotes` を修正
3. スクリプトを再実行
