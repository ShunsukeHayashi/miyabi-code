# The Sekai Stack (How it Works)
1. **Sense** — ByteDance/Larkカメラ、POS、勤怠、CRM、IoTが即日接続。PIIは自動マスキング。
2. **Foundry** — Sekai Foundryノーコード基盤でPoCフローを1営業日生成。Ω理論がエージェントの役割・Escalationを制御。
3. **Execute** — 業界別プレイブック＋Larkボット＋ダッシュボードで現場をナビゲート。AR-guided SOP（α）も紹介。
4. **Govern** — 期待値調整キット、監査ログ、権限管理で安心導入。週次Execレビューと成果レポートを標準提供。
