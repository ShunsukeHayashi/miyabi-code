全ドキュメントはGitで管理し、以下のルールに従う：

- **メジャー更新** (1.0 → 2.0): アーキテクチャ大幅変更時
- **マイナー更新** (1.0 → 1.1): 機能追加時
- **パッチ更新** (1.0.0 → 1.0.1): 誤字修正・微調整
