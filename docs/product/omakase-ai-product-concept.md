## 🎯 Executive Summary

### プロダクト名
**omakase.ai** - EC特化型音声AIショッピングアシスタント

### キャッチコピー
```
「人間のような接客を、全ての顧客に」
AI店員が24/7、ECサイトで最適な商品をご提案
```

### コアバリュー（3つ）

1. **EC販売特化の会話設計**
   - 商品推薦、カート誘導、クロスセル/アップセルに最適化された対話フロー

2. **日本市場特化の実装**
   - 日本語の自然な敬語対応
   - 日本のEC商習慣に最適化

3. **即導入可能なWidget統合**
   - 既存ECサイトに5分で実装
   - カート・商品APIとシームレス連携
   - カスタマイズ可能なUI

## 🏆 USP（Unique Selling Proposition）

### Core USP（1文）
```
日本のEC事業者向けに、
音声AIで購入転換率を平均35%向上させる、
EC特化型ショッピングアシスタント
```

### Differentiation（差別化要素）

#### 1. EC Sales Optimization（販売最適化）
**競合**: 汎用的な会話AI（VAPI, Pipecat）
**omakase.ai**: EC販売に特化した対話設計

| 機能 | 汎用AI | omakase.ai |
|------|--------|------------|
| 商品推薦アルゴリズム | ❌ なし | ✅ 購買データ学習 |
| カート誘導フロー | ❌ 手動設計 | ✅ テンプレート標準装備 |
| クロスセル/アップセル | ❌ 手動 | ✅ 自動提案 |
| 購入完了率 | 10-15% | **35-45%** |

#### 2. 日本市場特化
**競合**: グローバル製品（VAPI）
**omakase.ai**: 日本語・日本市場に最適化

- **言語**: 自然な日本語敬語（お客様対応レベル）
- **決済**: 日本の決済手段（代引き、コンビニ決済対応）
- **コンプライアンス**: 特定商取引法、個人情報保護法準拠
- **サポート**: 日本語サポート、国内時間帯対応

#### 3. 即導入可能なWidget
**競合**: 技術的統合が困難（Pipecat, Retell）
**omakase.ai**: 5分で実装完了

```javascript

<script src="https://cdn.omakase.ai/widget.js"></script>

```

- **技術要件**: なし（JavaScriptのみ）
- **API統合**: Shopify, BASE, EC-CUBEに対応
- **カスタマイズ**: ノーコードで見た目・挙動変更可能

## 🗺️ ポジショニングマップ

### 2軸ポジショニング

**X軸**: 汎用性 ←→ EC特化度
**Y軸**: セルフサービス ←→ フルサービス

```
                    フルサービス
                         ↑
                         |
         EC専門制作会社  |  Rep AI（高価格）
             ●           |      ●
                         |
                         |
             omakase.ai  |
                 🟢      |
                         |
    ─────────────────────┼─────────────────────→
    汎用性               |              EC特化度
                         |
         VAPI            |    Retell
          ●             |      ●
                         |
       Pipecat           |
          ●             |
                         |
                    セルフサービス
                         ↓
```

### ポジション解説

| 競合/自社 | 汎用性/特化度 | サービスレベル | 価格帯 |
|----------|--------------|---------------|--------|
| **omakase.ai** | EC特化（8/10） | ミッドタッチ（6/10） | ¥50,000-300,000/月 |
| VAPI | 汎用（3/10） | セルフサービス（2/10） | $45-500/月 |
| Pipecat | 汎用（2/10） | セルフサービス（1/10） | OSS無料 |
| Retell | やや特化（5/10） | セルフサービス（3/10） | $99-999/月 |
| Rep AI | EC特化（7/10） | フルサービス（9/10） | ¥500,000-2,000,000/月 |

### White Space（市場空白地帯）

**ターゲット**: EC特化度（7-8/10） × ミッドタッチ（5-7/10）

**理由**:
- 汎用AIは「EC販売」に最適化されていない
- フルサービス（Rep AI）は高価格で中小EC事業者が手を出せない
- **omakase.ai**: EC特化 × 手頃な価格 × 簡単導入 のスイートスポット

## 📈 Unit Economics（ユニットエコノミクス）

### SaaS Metrics

#### LTV（Life Time Value）計算

```
ARPU（月次）: ¥300,000（Proプラン平均）
Churn Rate: 3%/月（Year 2以降）
Gross Margin: 75%

LTV = ARPU ÷ Churn Rate × Gross Margin
    = ¥300,000 ÷ 3% × 75%
    = ¥7,500,000
```

#### CAC（Customer Acquisition Cost）計算

```
月次マーケティング費用: ¥3,000,000
月次営業費用: ¥2,000,000
新規顧客数: 10社/月

CAC = (¥3,000,000 + ¥2,000,000) ÷ 10
    = ¥500,000
```

#### LTV/CAC Ratio

```
LTV/CAC = ¥7,500,000 ÷ ¥500,000 = 15x
```

**評価**: ✅ **Excellent**（3x以上が健全、5x以上が優秀）

#### Payback Period（回収期間）

```
Payback = CAC ÷ (ARPU × Gross Margin)
        = ¥500,000 ÷ (¥300,000 × 75%)
        = 2.2ヶ月
```

**評価**: ✅ **Very Good**（12ヶ月以内が理想）

#### Rule of 40

```
Revenue Growth Rate（Year 2）: +300%
Profit Margin（Year 2）: -15%（投資フェーズ）

Rule of 40 = 300% + (-15%) = 285%
```

**評価**: ✅ **Excellent**（40%以上が健全）

3. **人手不足**: ECサポート人材不足が深刻化
4. **規制**: チャットボット規制強化でAI音声に注目

4. **経済不況**: EC投資予算削減リスク
