omakase.aiは、**4つの収益源を組み合わせたハイブリッドモデル**を採用します。

```

│              omakase.ai Revenue Model                    │

│                                                          │
└──────────────────────────────────────────────────────────┘
```

## 💰 プラン設計詳細

### 1. Starter プラン

**ターゲット**: 月商〜500万円の小規模EC事業者

| 項目 | 内容 |
|------|------|
| **月額料金** | ¥50,000 |
| **年額料金** | ¥540,000（¥45,000/月、10%オフ） |
| **通話時間** | 500分/月 |
| **超過料金** | ¥100/分 |
| **対象月商** | 〜¥5,000,000 |
| **想定CVR改善** | +50%（2% → 3%） |
| **想定ROI** | 10x |

#### 主要機能

✅ **含まれる機能**:
- 音声AIショッピングアシスタント（基本）
- EC特化プロンプトテンプレート（3業種）
- 商品推薦、質問応答、カート誘導
- Shopify連携

- カスタムプロンプト設計
- 専任CSM
- API優先サポート
- SLA保証

#### 価格設定根拠（Value-Based Pricing）

**顧客が得る価値（月商500万円の場合）**:

```
CVR向上効果:
- Before: 月商¥5,000,000、CVR 2%

- 売上増: ¥5,000,000 × 50% = ¥2,500,000/月

- Before: 有人対応 ¥300,000/月
- After:  AI自動化 ¥100,000/月
- 削減額: ¥200,000/月

合計価値: ¥2,700,000/月
```

**価格設定**:
- omakase.ai費用: ¥50,000/月

- **顧客ROI**: **54倍**

**リスク**:
- 測定の複雑性（他施策との切り分け）
- 売上変動リスク

## 💡 プライシング戦略

### 1. Value-Based Pricing（価値ベース価格設定）

**原則**: 顧客が得る価値の3〜5%をキャプチャー

```
顧客価値 = CVR向上効果 + コスト削減効果

Starterプラン:
価値¥2,700,000/月 → 価格¥50,000（1.85%）→ ROI 54x

Proプラン:
価値¥8,000,000/月 → 価格¥300,000（3.75%）→ ROI 27x

Enterpriseプラン:
価値¥35,600,000/月 → 価格¥800,000（2.25%）→ ROI 45x
```

**メリット**:
- 顧客は常に高いROIを実感
- 価格への抵抗感が低い
- 長期契約につながる

---

### 2. Freemium戦略（将来的に検討）

**現状**: Year 1-2は有料プランのみ

**Year 3以降の検討**:
- 無料プラン追加（月50分まで無料）
- 転換率: 3-5%想定
- 目的: ブランド認知度向上、ユーザー数拡大

---

### 3. Usage-Based Upsell（利用量ベースアップセル）

**トリガー**: 通話時間が80%超過時

```
例: Starterプラン顧客が400分使用（80%）

自動メール:
「今月の通話時間が400分に達しました。
来月は超過料金が発生する可能性があります。

Proプランにアップグレードすると、
月額¥250,000の追加で3,000分まで使用可能です。」
```

**自動化**: 利用率80%で自動メール送信

---

### 4. Anchor Pricing（アンカー価格設定）

**表示順序**: 高 → 中 → 低（アンカー効果）

```
┌─────────────────────────────────────────┐
│  Enterprise  │  Pro（推奨）  │  Starter  │
│  ¥800,000〜  │  ¥300,000    │  ¥50,000  │
└─────────────────────────────────────────┘
```

**効果**:
- Proプランが「ちょうど良い価格」に見える
- Enterprise価格がアンカーとなり、Pro選択率向上

---

### 5. Annual Discount（年次割引）

**割引率**: 10%

**目的**:
- キャッシュフロー改善
- チャーン率低下
- 長期コミットメント

**Year 1目標**: 年次契約率30%

## 📈 LTV/CAC分析

### LTV（Life Time Value）詳細計算

#### Proプラン想定

```
ARPU（月次）: ¥300,000
Gross Margin: 75%
Churn Rate: 3%/月（Year 2以降）

LTV = ARPU × Gross Margin ÷ Churn Rate
    = ¥300,000 × 75% ÷ 3%
    = ¥7,500,000
```

### CAC（Customer Acquisition Cost）詳細計算

```
月次マーケティング費用:
- Google広告: ¥1,500,000
- SEO/コンテンツ: ¥500,000
- ウェビナー: ¥200,000
- メディア掲載: ¥300,000
- イベント: ¥200,000
合計: ¥2,700,000

月次営業費用:
- セールス人件費（3名）: ¥1,500,000
- 営業経費: ¥300,000
合計: ¥1,800,000

新規顧客数: 10社/月（Year 2想定）

CAC = (¥2,700,000 + ¥1,800,000) ÷ 10
    = ¥450,000
```

### LTV/CAC Ratio

```
LTV/CAC = ¥7,500,000 ÷ ¥450,000 = 16.7x
```

**評価**: ✅ **Excellent**（5x以上が優秀）

### Payback Period（回収期間）

```
Payback = CAC ÷ (ARPU × Gross Margin)
        = ¥450,000 ÷ (¥300,000 × 75%)
        = 2.0ヶ月
```

**評価**: ✅ **Very Good**（12ヶ月以内が理想）
