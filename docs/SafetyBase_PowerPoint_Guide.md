# SafetyBase ピッチデック PowerPoint変換ガイド

**目的**: Markdownピッチデックをデザイン付きPowerPointに変換
**作成日**: 2025年11月21日

---

## 1. 推奨変換ツール

### Option A: Marp（推奨・無料）

Markdownから直接PPTXに変換できる。

```bash
# インストール
npm install -g @marp-team/marp-cli

# 変換コマンド
marp SafetyBase_PitchDeck.md --pptx -o SafetyBase_PitchDeck.pptx
```

### Option B: Slidev

```bash
npm install -g @slidev/cli
slidev export SafetyBase_PitchDeck.md --format pptx
```

### Option C: Pandoc

```bash
pandoc SafetyBase_PitchDeck.md -o SafetyBase_PitchDeck.pptx
```

### Option D: 手動変換（デザイン重視）

Markdownを参照しながら、PowerPoint/Keynote/Google Slidesで手動作成。

## 3. スライド別デザイン指示

### Slide 1: タイトル

**レイアウト**: 中央配置
**要素**:
- ロゴ（大）：中央上部
- タイトル：「SafetyBase」（大きく、Primary色）
- サブタイトル：キャッチコピー
- 日付・調達額：下部に小さく

**背景**: グラデーション（白→薄いグリーン）

---

### Slide 2: 課題

**レイアウト**: 左右2カラム
**左側**: 課題リスト（アイコン付き）
**右側**: インパクトある数字（「毎日2時間」等を大きく）

**背景**: 薄いグレー
**アクセント**: 問題を示す赤/オレンジのアイコン

---

### Slide 3: ソリューション

**レイアウト**: 中央集中
**要素**:
- 8サブシステムのアイコングリッド
- 中央に「SafetyBase」ロゴ
- 接続線で統合を表現

**アニメーション**: 8つのアイコンが中央に集まるアニメーション

---

### Slide 4: デモ / 製品画面

**レイアウト**: 製品スクリーンショット中心
**要素**:
- モックアップフレーム内にUI画面
- 吹き出しで主要機能を説明

**注意**: 実際の製品画面を使用

---

### Slide 5: 市場規模

**レイアウト**: 同心円グラフ
**要素**:
- TAM > SAM > SOM の同心円
- 各円に金額を明記

**カラー**: グリーンのグラデーション（外側ほど薄く）

---

### Slide 6: ビジネスモデル

**レイアウト**: 上下分割
**上部**: 価格表（3プラン）
**下部**: KPI指標（LTV/CAC、解約率等）

**強調**: 数字は大きく、色でハイライト

---

### Slide 7: トラクション

**レイアウト**: タイムライン + 成長グラフ
**左側**: マイルストーンタイムライン
**右側**: ARR成長の折れ線グラフ

**アニメーション**: 順番に表示

---

### Slide 8: 競合分析

**レイアウト**: 比較表
**要素**:
- 4列の比較表（SafetyBase vs 競合3社）
- チェックマーク/バツで視覚化
- SafetyBaseの列をハイライト

**カラー**: SafetyBase列はPrimary色背景

---

### Slide 9: Go-to-Market戦略

**レイアウト**: 3フェーズのフロー図
**要素**:
- 3つの矢印で接続されたボックス
- 各フェーズの目標と主要施策

**アニメーション**: 左から右へ順番に表示

---

### Slide 10: チーム

**レイアウト**: 4人のグリッド
**要素**:
- 顔写真（または高品質アイコン）
- 名前、役職
- 1行の実績ハイライト

**背景**: プロフェッショナルなグレー

---

### Slide 11: 財務計画

**レイアウト**: データ中心
**上部**: 5年間の主要指標表
**下部**: ARR成長の棒グラフ

**強調**: 3年目の数字（11.5億円）を大きく

---

### Slide 12: 資金調達

**レイアウト**: 中央配置
**要素**:
- 大きく「2億円」
- 円グラフで使途内訳

**カラー**: Accent色で強調

---

### Slide 13: Why Now

**レイアウト**: 4象限またはアイコンリスト
**要素**:
- 4つの追い風をアイコン付きで
- 各項目に簡潔な説明

**アニメーション**: 1つずつフェードイン

---

### Slide 14: Next Steps

**レイアウト**: チェックリスト形式
**要素**:
- 3ヶ月のマイルストーン
- 調達後の計画

**デザイン**: アクションを示すチェックマークアイコン

---

### Slide 15: Contact

**レイアウト**: シンプル、中央配置
**要素**:
- 会社名
- 連絡先（メール、電話）
- QRコード（オプション）

**背景**: クリーンな白
