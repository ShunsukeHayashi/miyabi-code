## カスタマージャーニー概要

このドキュメントでは、3つの主要ペルソナのカスタマージャーニーを詳細にマッピングします。

1. **佐藤ゆめ（VTuber準備者）** - メインターゲット
2. **桜井さくら（同人クリエイター）** - サブターゲット
3. **田中健太（YouTuberクリエイター）** - サブターゲット

各ステージで、顧客の行動、思考、感情、タッチポイント、課題、機会を明確化します。

---

## ジャーニーステージ定義

| ステージ | 説明 | ゴール |
|---------|------|--------|
| **1. 認知（Awareness）** | 課題を認識し、解決策を探し始める | Miyabi Character Studioの存在を知る |
| **2. 興味（Interest）** | 製品に興味を持ち、詳細を調べる | 「これは使えそう」と感じる |
| **3. 検討（Consideration）** | 他の選択肢と比較検討する | 「これが最適」と確信する |
| **4. 購入（Purchase）** | 無料トライアルまたは有料プラン購入 | スムーズに購入完了 |
| **5. 利用（Retention）** | 実際にツールを使って価値を実感 | 継続利用を決意 |
| **6. 推奨（Advocacy）** | SNSでシェア、友人に紹介 | アンバサダーになる |
