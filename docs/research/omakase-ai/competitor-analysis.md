# omakase.ai - 競合分析レポート

**調査日**: 2025-12-06
**分析対象**: EC向け音声AIアシスタント関連プレイヤー（25社）
**担当Agent**: MarketResearchAgent (市 いちばさん 📈)

---

## エグゼクティブサマリー

EC音声アシスタント市場における競合状況を25社分析した結果、以下の知見を得た:

1. **直接競合（EC特化音声アシスタント）**: 3社のみ（Rep AI、Pronto AI、Relish AI）
2. **間接競合（汎用音声AIプラットフォーム）**: 12社（VAPI、Retell、Bland AI等）
3. **潜在競合（ECチャットボット→音声化）**: 10社（Tidio、Conversagent等）

**重要な発見**:
- 日本市場における「EC特化音声アシスタント」は**ほぼ未開拓**（ブルーオーシャン）
- グローバルではVAPI等の汎用プラットフォームが主流だが、EC特化型は少数
- omakase.aiの競合優位性は「**日本語×EC特化×プロンプト最適化**」の3点セット
