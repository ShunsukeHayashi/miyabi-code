## 1. 動画概要

| 項目 | 内容 |
|------|------|
| タイトル | SafetyBase: 中小建設会社のためのオールインワンDXソリューション |
| 尺 | 約3分 |
| コアメッセージ | 「エンタープライズ品質のDXを中小企業に。書類80%削減、コスト50%削減」 |

### ターゲット視聴者

- **見込み顧客**: 中小建設会社経営者、現場所長（効率化を求める）

## 2. 詳細スクリプト

| 時間 | 映像 | ナレーション |

| 0:00 | 散らかった事務所、山積みの紙書類（グリーンファイル）、21時を指す時計を見つめる疲れた管理者 | 建設業界は今、危機に直面しています。残業規制の強化、慢性的な人手不足。 |

### Scene 2: ソリューション紹介（0:25-0:50）

| 時間 | 映像 | ナレーション |

### Scene 3: デモ Part 1 - ワンクリック書類作成（0:50-1:20）

| 1:10 | テキスト：「ワンクリック書類作成」 | プロジェクトと作業員を選ぶだけ。既存のマスタデータから全項目を自動入力。クリック、印刷、完了です。 |

## 3. 制作ガイドライン

### 3.1 ビジュアルスタイル

| 要素 | ガイドライン |
|------|-------------|
| **UI撮影** | 画面収録時は特定のボタンやフィールドにズームイン。カーソルハイライター（黄色の円）を使用 |
| **配色** | ブランドカラー（セーフティグリーン、ネイビーブルー、ホワイト）を使用 |
| **素材映像** | 「モダンな」建設作業員（タブレット/スマホを使用）の高品質映像。ハンマーを振る映像だけは避ける |

### 3.2 オーディオ戦略

| 要素 | ガイドライン |
|------|-------------|
| **ナレーション** | 権威ある温かみのある声。日本市場向けには明瞭な標準語 |
| **BGM 0:00-0:25** | 緊張感、少し切迫した曲調（問題を強調） |
| **BGM 0:25-終了** | アップテンポ、テクノロジー感、躍動感（解決策を強調） |

### 3.3 投資家向けポイント

| ポイント | 演出方法 |
|----------|----------|
| **ノーコード機能** | 視覚的に強調。高い解約率の原因である「硬直性」の解消を証明 |
| **8サブシステム** | 「ハブ」として視覚化。ポイントソリューションではなくプラットフォームであることを示す |

### 3.4 顧客向けチュートリアル要素

| ポイント | 演出方法 |
|----------|----------|
| **Show, Don't Tell** | 「ワンクリック」と言うときは、即座にそのクリックを見せる |
| **モバイル画面** | 建設がターゲットなのでモバイル表示は必須。レスポンシブで読みやすい画面を |

## 4. 制作チェックリスト

### Pre-Production

- [ ] スクリプトをプロダクトチームと確認（機能の正確性）
- [ ] ダミーデータ作成（プロジェクトA、作業員田中等 - プライバシー用）
- [ ] ナレーター手配（プロの声優またはAI音声ツール）

### Production

- [ ] 「ハッピーパス」の画面収録（書類作成、モバイル日報）
- [ ] BGM選定・ライセンス取得
- [ ] グラフィック作成（50%低コスト比較グラフ等）

### Post-Production

- [ ] 画面収録と音声の同期
- [ ] テキストオーバーレイ追加
- [ ] 字幕作成（音声オフ視聴者向け）
- [ ] 最終レビュー（音声なしで視聴してメッセージが伝わるか確認）
