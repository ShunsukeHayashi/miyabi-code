# NEXYZ.Group 包括的ビジネス分析レポート

**作成日**: 2025-10-29
**分析対象**: 株式会社NEXYZ.Group（東証スタンダード 4346）
**Webサイト**: https://www.nexyzgroup.jp/

---

## エグゼクティブサマリー

NEXYZ.Groupは、エンベデッド・ファイナンス（組込型金融）を活用した革新的な「初期投資ゼロ円」ビジネスモデルで急成長中の東京の持株会社。主力事業「ネクシーズZERO」では、LED照明・業務用冷蔵庫・空調設備を月額定額制で提供し、107,391件以上の導入実績を誇る。2024年9月期の営業利益は前年比55.2%増と高成長を維持。

**Key Highlights**:
- 売上高: ¥24.5B (+11.6% YoY)
- 営業利益: ¥1.2B (+55.2% YoY)
- 主力セグメント: エンベデッド・ファイナンス事業（売上構成比80.4%）
- 顧客獲得効率: リピート・紹介・Web経由が90%（新規営業は11.5%のみ）
- 新規事業: ACCEL JAPAN（芸能人活用の定額広告サービス）

## 3. ビジネスモデル詳細分析

### 3.1 コアビジネス: ネクシーズZERO

**Value Proposition**:
> "初期投資ゼロ円・月額定額制で、最新の業務用設備を導入。現地調査から施工、アフターケアまで全てワンストップで提供"

**ビジネスモデル図**:
```
┌─────────────────────────────────────────────────┐
│           ネクシーズZERO ビジネスフロー          │
└─────────────────────────────────────────────────┘

[顧客企業] ←─┐
   │          │
   │ ①問い合わせ（Web/紹介/既存顧客）
   ↓          │
[NEXYZ.Facilities] ──┐
   │                  │
   │ ②現地調査        │ ⑦アフターサービス
   │ ③設備選定        │   メンテナンス
   │ ④施工           │   修理・交換
   ↓                  │
[提携メーカー200社]   │
   │                  │
   │ ⑤製品供給        │
   ↓                  │
[顧客企業] ←─────┘
   │
   │ ⑥月額支払い（固定）
   ↓
[NEXYZ.Group]
   │
   │ 収益源:
   │ - 月額利用料
   │ - 初期セットアップフィー
   │ - メンテナンス収益
   │ - 設備更新時の追加売上
   └─→ ムーディーズAaa(sf)格付け取得
        （グリーンファイナンス）
```

**対象設備カテゴリー**:
1. **LED照明**
   - 店舗・施設まるごとLED化
   - 電気代削減効果: 月額費用をカバー可能
   - 提携: パナソニック、東芝、アイリスオーヤマ等

2. **業務用冷蔵庫・冷凍庫**
   - 飲食店・小売店向け
   - 省エネモデル
   - 提携: ホシザキ、フクシマガリレイ等

3. **業務用エアコン**
   - オフィス・店舗向け
   - 省エネ効果
   - 提携: ダイキン、三菱電機等

**提携メーカー**: 200社以上、製品ラインナップ: 500,000点以上

### 3.5 競争優位性（5つの強み）

#### 1. ゼロ円導入モデルの先駆者優位性
- 2010年代前半から「初期投資ゼロ円」を展開
- 累計107,391件の導入実績 → 市場認知度とブランド確立
- ムーディーズ最高格付け「Aaa(sf)」取得（信頼性）

#### 2. マルチカテゴリー × ワンストップ提供
- 競合は単一カテゴリー（LED専業、厨房機器専業等）
- NEXYZはLED+空調+厨房を統合提供 → クロスセル強い
- 200社のメーカー提携 × 500,000製品 → 選択肢の広さ

#### 3. 高効率な顧客獲得構造
- 新規営業わずか11.5%で成長継続
- 90%がリピート・紹介・Web → CAC（顧客獲得コスト）低い
- LTV（顧客生涯価値）高い（追加設備導入、長期契約）

#### 4. エンベデッド・ファイナンス × グリーン投資
- 環境省「エコ・ファースト企業」認定（2020年）
- ESG投資需要に対応 → 大口資金調達可能
- LED等の省エネ設備 → 顧客のSDGs貢献支援

#### 5. 新規事業創出力（ACCEL JAPAN等）
- 「ゼロ円モデル」を芸能人広告に応用
- 既存の成功パターンを水平展開
- ブランジスタ（上場子会社）のメディアノウハウ活用
