# 05 統合報告書テンプレート

目的: 全ドメイン（経済・医療・教育・行政・分析・安全/倫理）の成果・課題・意思決定を統合し、次期アクションを合意するための定期レポート雛形。

## 1. 報告サマリー
- 期間: YYYY-MM-DD 〜 YYYY-MM-DD
- 総合ステータス: 🟢/🟡/🔴
- 主要トピック（3〜5件）

## 2. ドメイン別進捗
- 経済: 主要KPI、成果、課題、リスク、依存関係
- 医療: 主要KPI、成果、課題、リスク、依存関係
- 教育: 同上
- 行政: 同上
- 分析: 同上
- 安全/倫理: 監査結果、指摘、是正状況

## 3. KPI/メトリクス
- 住民支持率/参加率
- 財政: 収入/支出/自立度、予備費
- サービスSLA: 稼働率、P99、重大インシデント件数
- 安全/セキュリティ: MTTD/MTTC、重大アラート
- 法務/コンプライアンス: 指摘件数、是正率

## 4. リスク・課題・対応
- 新規/継続リスク一覧（発生日、重大度、担当、次アクション、期限）
- ブロッカーと解消計画

## 5. 意思決定事項
- 決定内容、根拠データ、決定者、施行日、影響範囲
- 必要な追認（住民/評議会）と予定日

## 6. 次期アクションプラン
- 1〜4週間の短期アクション（担当・期限・期待成果）
- 1四半期の重点テーマ

## 7. 依存関係・リクエスト
- 他ドメイン/外部への依頼事項、必要期日、前提条件

## 8. 付録
- データ/ダッシュボードリンク
- 監査ログ・RCA要約
- 変更履歴（版数/日付/編集者）
