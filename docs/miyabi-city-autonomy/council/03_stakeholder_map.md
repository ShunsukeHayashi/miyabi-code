# 03 ステークホルダーマップ

## 内部ステークホルダー
- 住民（100人コア＋周辺住民）: 生活の質、安全、費用負担、意思決定参加。
- 評議会/執行部: 方針・予算・リスク管理の責任主体。
- 職員・ボランティア: 運用・サービス提供・現場改善。
- ドメイン委員会（経済・医療・教育・行政・分析・安全/倫理）: SLA/KPI達成、調整。

## 外部ステークホルダー
- 国・都道府県・近隣自治体: 法的適合性、治安、インフラ相互接続、財政監督。
- 事業者/投資家: 収益性・安定性・規制リスク・ブランド。
- 教育機関/医療機関/研究機関: 実証・人材・共同研究。
- 市民社会/メディア: 透明性・説明責任・倫理。
- インフラ事業者（電力・通信・物流）: 契約安定、需要予測、相互依存リスク。

## 利害と対応戦略（例）
- 国・自治体: 合法性/治安確保 → 早期協議・法務レビュー・透明な財政/治安データ共有。
- 近隣住民/企業: 外部性（交通/騒音/需給） → 影響評価と緩和策、定期ブリーフィング。
- 投資家: 規制・政治リスク → 段階投資、エスクロー/ガバナンス条項、リスク開示。
- インフラ事業者: 料金滞納/契約破綻 → 長期契約、予備費、需給共同計画。
- 住民: 負担と権利のバランス → 参加型予算、料金/税の透明化、フィードバックサイクル。

## コミュニケーション計画
- 定期: 月次オープンブリーフィング、四半期統合報告、年次レビュー。
- 緊急: 公式チャネル（SMS/アプリ/メール/掲示）、30分以内の一次報、事後レビュー公開。
- 住民参加: 住民総会、オンライン投票、パブコメ、ワークショップ。
- 対外: 行政向け協議資料、投資家アップデート、メディアQ&A、研究連携報告。
