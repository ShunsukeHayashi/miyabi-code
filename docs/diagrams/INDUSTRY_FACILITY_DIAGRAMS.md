# 設備工事業 業務フロー PlantUML ダイアグラム

## 概要

このディレクトリには、設備工事業（空調・給排水設備工事）の各部署における業務フローを示すPlantUMLダイアグラムが含まれています。

**業界**: 設備工事業（HVAC・給排水設備）
**フォーマット**: PlantUML + cerulean-outline テーマ
**言語**: 日本語

## ダイアグラムの特徴

### Swimlane（泳道図）を使用した表現
- 各部署または役職ごとのプロセスを視覚的に整理
- 部署間のインターアクションを明確に表現
- 意思決定ポイント（条件分岐）を含む

### 実装されたビジネスロジック
- **営業プロセス**: 見積作成から受注管理まで
- **設計プロセス**: 現場調査から詳細設計・検査まで
- **施工プロセス**: 工事計画から竣工引渡しまで
- **メンテナンス**: 定期点検から故障対応まで
- **安全管理**: リスク評価から事故対応まで
- **財務管理**: 原価計算から決算報告まで
- **人事総務**: 採用から文書管理まで

### 現実的な業務詳細
- 各工種の具体的な名称（配管工、電気工、ダクト工）
- チェックリスト項目の記載
- 法規対応（労災報告、建設業協会報告など）
- リスク評価プロセス（4M4E分析など）

## 使用方法

### PlantUMLの描画
これらのファイルはPlantUMLで描画できます：

```bash
# インストール（Macの場合）
brew install plantuml

# PNG画像に変換
plantuml -Tpng industry_facility_営業部.puml

# SVG形式に変換
plantuml -Tsvg industry_facility_営業部.puml
```

### VS Code での表示
- PlantUML拡張機能をインストール
- ファイルを右クリック → "Show PlantUML Preview"

### Web での表示
- https://www.plantuml.com/plantuml/uml/
- PlantUMLコードをペーストして表示

## カスタマイズポイント

各ファイルは以下の要素をカスタマイズ可能です：

1. **部署・役職名**: 実際の組織構造に合わせて変更
2. **プロセス内容**: 社内の実際のフローに合わせて調整
3. **チェックリスト**: 業界・企業固有の項目に変更
4. **条件分岐**: 実際の判定基準に合わせて修正
5. **外部連携**: 協力業者や顧客とのインターアクション
