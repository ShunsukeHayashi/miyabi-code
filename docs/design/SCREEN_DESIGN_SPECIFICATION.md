## State Management

### Global State (Zustand stores)

1. **authStore** (`src/stores/authStore.ts`)
   - `token`, `accessToken`, `user`, `isAuthenticated`, `isLoading`
   - Actions: `setAuth()`, `logout()`, `clearAuth()`

2. **websocketStore** (TBD)
   - `connected`, `events`, `subscriptions`
   - Actions: `connect()`, `disconnect()`, `subscribe()`, `send()`

3. **agentExecutionStore** (TBD)
   - `executions[]`, `activeExecution`
   - Actions: `addExecution()`, `updateExecution()`, `removeExecution()`

### Server State (TanStack Query)

**Query Keys**:
- `['repositories']` - User's connected repositories
- `['repositories', id, 'issues']` - Issues for a repository
- `['agents', executionId]` - Agent execution detail
- `['workflows']` - User's workflows
- `['dashboard', 'summary']` - Dashboard summary stats

**Mutations**:
- `executeAgent` - POST `/api/agents/execute`
- `connectRepository` - POST `/api/repositories`
- `createWorkflow` - POST `/api/workflows`
