## その場で実演！Miyabi LPGenAgentの威力

「じゃあ、今からこのデザインを参考に、新しいLPを作ってみましょうか」

僕はMacを開いて、Miyabi CLIを起動しました。

```bash
miyabi agent run lp-gen --reference https://happytry-lp.site?vos=meta
```

すると、Miyabiが自動的に以下を実行しました：

### ステップ1: デザイン分析（30秒）
MiyabiのLPGenAgentが、参考URLのサイトを自動で分析します。

- レイアウト構造（10セクション構成）
- カラースキーム（白背景 + アクセントカラー）
- レスポンシブデザインのブレークポイント
- CTA（Call To Action）ボタンの配置パターン

### ステップ2: 対話形式でのカスタマイズ（3分）
Miyabiが寿司次郎さんに質問します：

**Miyabi**: 「サービス名を教えてください」
**寿司次郎さん**: 「『寿司マーケティング講座』です」

**Miyabi**: 「ターゲット顧客は？」
**寿司次郎さん**: 「飲食店経営者とコンテンツクリエイター」

**Miyabi**: 「キャッチコピーは？」
**寿司次郎さん**: 「寿司職人が教える、心を掴むストーリーテリング術」

### ステップ3: LP自動生成（2分）
Miyabiが以下を自動で生成：

- **HTML**: TailwindCSSベースのレスポンシブLP
- **10セクション構成**:
  1. ヒーローセクション（メインビジュアル + キャッチコピー）
  2. 課題提示（ターゲットの悩み）
  3. 解決策（サービス紹介）
  4. ユーザーの声（信頼性強化）
  5. サービス詳細（3つの特徴）
  6. 価格プラン
  7. メディア掲載実績
  8. FAQ
  9. 講師プロフィール
  10. 最終CTA（申込ボタン）

- **レスポンシブ対応**: PC・タブレット・スマホで最適表示
- **アクセシビリティ**: ARIA属性、セマンティックHTML対応

### ステップ4: プレビュー確認（1分）
ローカルサーバーで即座にプレビュー表示。

```bash
# Miyabiが自動でローカルサーバー起動
miyabi preview lp-gen-output/index.html
```

ブラウザが開いて、完成したLPが表示されました。

## MiyabiのLPGenAgentとは？

寿司次郎さんが体験した**LPGenAgent**は、Miyabiフレームワークの15個のBusiness Agentの1つです。

### 主な機能

1. **参考URL解析**: 既存のLPデザインを自動分析
2. **対話型カスタマイズ**: 必要な情報を質問形式で収集
3. **自動コード生成**: HTML + TailwindCSSで本格LP生成
4. **レスポンシブ対応**: モバイルファーストデザイン
5. **SEO最適化**: メタタグ、OGP、構造化データ自動挿入
6. **再利用可能**: コンテンツJSONで簡単編集

### 対応デザインパターン

- **VSL型LP**: ビデオセールスレター特化
- **リストビルディング型**: メールアドレス収集特化
- **ウェビナー型**: イベント申込特化
- **SaaS型**: 無料トライアル訴求
- **Eコマース型**: 商品販売特化

## よくある質問（FAQ）

### Q1: コーディングスキルは必要ですか？
A1: いいえ、一切不要です。対話形式で質問に答えるだけでLPが完成します。

### Q2: 商用利用は可能ですか？
A2: プロプラン以上で商用利用が可能です。スタータープランは個人利用のみです。

### Q3: 既存のLPをリニューアルできますか？
A3: はい、既存LPのURLを参考URLとして指定することで、デザインを踏襲しつつコンテンツを刷新できます。

### Q4: 生成されたコードは編集できますか？
A4: はい、生成されたHTMLは完全にオープンで、自由に編集可能です。

### Q5: WordPress等のCMSとの統合は？
A5: 現在、WordPress、Wix、Shopifyへのエクスポート機能を開発中です（2025年12月リリース予定）。
