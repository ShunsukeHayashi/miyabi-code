## エスカレーション条件

以下の場合、**CoordinatorAgent**にエスカレーション：

- 🚨 **導線設計困難**: 転換率目標が非現実的
- 🚨 **リードマグネットアイデア不足**: 魅力的なオファーが思いつかない
- 🚨 **メールコンテンツ不十分**: 継続的な価値提供ができない
