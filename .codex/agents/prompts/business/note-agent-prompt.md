#### セクション6: Amazonアソシエイトリンク挿入

**最重要**: 記事と相性がいい商品を必ず選定し、Amazonアソシエイトタグ`shuhayas-22`を含むURLを生成してください。これは**絶対に間違えてはいけません**。

**実行内容**:
1. 記事内容から関連性の高い商品を3〜5個選定
2. 各商品のAmazon URLを取得
3. アソシエイトタグを追加してリンクを生成
4. CTAとしてユーザーをクリックベイト

**Amazonアソシエイトリンク生成手順**:
```
1. Amazon.co.jpで商品を検索
2. 商品ページのURLを取得
   例: https://www.amazon.co.jp/dp/B08XYZ1234/

3. アソシエイトタグを追加
   https://www.amazon.co.jp/dp/B08XYZ1234/?tag=shuhayas-22

4. 完全なURLフォーマット
   https://www.amazon.co.jp/dp/[ASIN]/?tag=shuhayas-22
```

**挿入テンプレート**:
```markdown
## 📚 おすすめの関連商品

### 1. [商品名]

[商品の説明文 - なぜこの記事の読者にとって役立つのかを説明]

[商品画像があれば]
[--IMAGE--]

**Amazonで詳しく見る**: [商品名](https://www.amazon.co.jp/dp/[ASIN]/?tag=shuhayas-22)

---

### 2. [商品名]

[商品の説明文]

[--IMAGE--]

**Amazonで詳しく見る**: [商品名](https://www.amazon.co.jp/dp/[ASIN]/?tag=shuhayas-22)
```

**挿入位置**:
- まとめセクションの後
- CTA（フォロー促進）の前
- 記事の最後（最も効果的）

**クリックベイトテクニック**:
```
✅ 良い例:
「この記事を読んで実践したいと思ったあなたに、さらに学びを深める書籍をご紹介します。私も実際に読んで役立った3冊です。」

❌ 悪い例:
「この商品を買ってください。」「おすすめです。」
```

**注意事項**:
- ❌ 記事内容と無関係な商品を紹介しない
- ✅ 読者の悩みを解決する商品を選ぶ
- ✅ 自然な流れでリンクを挿入
- ✅ 商品説明は50〜100文字程度

- 800〜1200文字ごとに1枚

**プレースホルダー後の処理**:
