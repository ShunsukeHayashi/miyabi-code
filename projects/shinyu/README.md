# Shinyu（真由）- 統合占いアプリ

**作成日**: 2025-10-25
**ステータス**: 計画中
**優先度**: P1-High

---

## 🎯 プロジェクト概要

AI駆動の統合占いアプリ「Shinyu（真由）」を開発する。

### コンセプト
複数の占術（タロット、易経、西洋占星術等）を統合し、AIが総合的に分析・アドバイスを提供する占いプラットフォーム。

---

## 📋 主要機能（想定）

### Phase 1: MVP
1. **タロット占い** - 78枚のタロットカード
2. **AI解釈** - GPT-4による総合分析
3. **Web UI** - Next.js シンプルインターフェース

### Phase 2: 拡張
4. **易経占い** - 64卦
5. **西洋占星術** - ホロスコープ生成
6. **相性診断** - 人間関係分析

### Phase 3: マネタイズ
7. **有料プラン** - 詳細鑑定
8. **サブスクリプション** - 月額プラン
9. **API提供** - B2B展開

## 🏗️ アーキテクチャ（想定）

### 技術スタック
- **フロントエンド**: Next.js 16 + TypeScript
- **バックエンド**: Rust (Axum) or Node.js
- **LLM**: OpenAI GPT-4o
