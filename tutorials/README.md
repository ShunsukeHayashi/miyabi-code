## üõ†Ô∏è Setup Instructions

Before starting the tutorials:

```bash
# 1. Clone Miyabi repository
git clone https://github.com/ShunsukeHayashi/Miyabi.git
cd Miyabi

# 2. Install Rust (if not already installed)
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# 3. Build Miyabi
cargo build --release

# 4. Set up environment variables
cp .env.example .env
# Edit .env with your GitHub token and API keys

# 5. Verify installation
./target/release/miyabi --version
```

**Detailed setup**: See [Tutorial 1: Getting Started](./01-getting-started.md)
