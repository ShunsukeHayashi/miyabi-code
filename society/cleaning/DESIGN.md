# Cleaning Society 設計書

## 概要
ビル・オフィス・店舗・マンションの清掃業務を完全自動化するAIエージェント群

## エージェント構成（8体）

### 1. 案件くん（Project Agent）
**役割**: 清掃案件の受付・見積・契約管理
**機能**:
- 問い合わせ自動対応
- 現地調査スケジュール
- 見積自動作成（面積・汚れ・頻度から）
- 契約書自動生成
**自動化効果**: 見積作成時間 2時間→10分（-92%）

### 2. スケジュールくん（Schedule Agent）
**役割**: 清掃スタッフのシフト・配置最適化
**機能**:
- AIシフト自動作成
- スタッフスキルマッチング
- ルート最適化（移動時間最小化）
- 急な欠勤対応（代替自動手配）
**自動化効果**: シフト作成 8時間→30分（-94%）

### 3. 品質くん（Quality Agent）
**役割**: 清掃品質の管理・チェック
**機能**:
- チェックリスト自動生成
- 写真AI判定（清掃前後比較）
- 品質スコアリング
- クレーム予防アラート
**自動化効果**: 品質チェック時間 -75%、クレーム -60%

### 4. 資材くん（Supply Agent）
**役割**: 清掃資材・消耗品の在庫管理
**機能**:
- 自動発注（閾値管理）
- 使用量予測
- コスト最適化
- 仕入先比較
**自動化効果**: 在庫切れゼロ、資材コスト -15%

### 5. スタッフくん（Staff Agent）
**役割**: 清掃スタッフの採用・教育・管理
**機能**:
- 採用応募自動対応
- スキルマップ管理
- 教育コンテンツ配信
- 勤怠・給与連携
**自動化効果**: 採用リードタイム -50%、離職率 -20%

### 6. 顧客くん（Customer Agent）
**役割**: 顧客対応・リレーション管理
**機能**:
- 定期報告自動生成
- 満足度調査
- 追加提案（ワックス・エアコン等）
- 契約更新管理
**自動化効果**: 顧客対応工数 -70%、追加受注 +30%

### 7. 分析くん（Analytics Agent）
**役割**: 経営分析・レポート
**機能**:
- 案件別収益分析
- スタッフ生産性分析
- エリア別需要予測
- 経営ダッシュボード
**自動化効果**: レポート作成 -99%

### 8. 清掃統括くん（Cleaning Director）
**役割**: 全体最適化・意思決定支援
**機能**:
- 7エージェント統合管理
- 異常検知・エスカレーション
- 経営判断サポート
- 新規事業提案

## 技術実装

### MCP Tools
- `cleaning_estimate`: 見積自動作成
- `cleaning_schedule`: シフト最適化
- `cleaning_quality_check`: 品質AI判定
- `cleaning_supply_order`: 資材発注
- `cleaning_report`: 報告書生成
