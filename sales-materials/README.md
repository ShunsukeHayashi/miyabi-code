- `faq.md` - よくある質問集（15項目）

## 🚀 使い方

### セミナー後フォローアップ（Day 0-3）

**ステップ1**: サンキューメール送信
```
件名: 【BytePlus Video API】セミナーご参加ありがとうございました

本文:
- セミナー録画リンク
- 無料クーポンコード（50万トークン）
- ROI試算シート（tools/roi-calculator.md）
```

**ステップ2**: 個別相談予約案内（Day 1）
```
- 提案書テンプレート（proposals/template.md）をカスタマイズ
- 御社専用のROI試算を作成
```

### 商談フェーズ（Week 5-8）

**ステップ1**: 提案書提出
```
- proposals/template.md をベースに御社用にカスタマイズ
- 現状分析、ROI試算を具体的な数値で記載
```

**ステップ2**: 稟議書サポート
```
- 契約書テンプレート（contracts/）を提供
- FAQ（faq/faq.md）で懸念点を事前に払拭
```

### 契約フェーズ（Week 9-12）

**ステップ1**: 契約書締結
```
- contracts/api-license.md
- contracts/nda.md
- contracts/sla.md
```

**ステップ2**: オンボーディング開始
```
- onboarding/ のチェックリストに従って導入支援
```

## 🎯 制約条件（ByteDance会議より）

### ✅ 使用可能
- BytePlus公式パートナー名
- API技術仕様
- 料金情報
- サポート体制

### ❌ 使用不可
- ByteDance名（親会社名）
- TikTok関連の言及
- 他社名を明記した比較資料

### ⚠️ 注意事項
- 機能比較は「一般的なAPI」との比較のみ
- 競合他社名は記載しない

## 📝 カスタマイズ方法

### 提案書のカスタマイズ

1. **表紙**
   - `【御社名】 様` → 実際の御社名に置換
   - `YYYY年MM月DD日` → 提案日に置換
   - `[担当者名]` → 実際の担当者名に置換

2. **現状分析（第2条）**
   - ヒアリング結果を元に記載
   - 月間動画本数、外注費等を具体的数値で記載

3. **ROI試算（第6条）**
   - `roi-calculator.md` を使用して試算
   - 具体的な削減額、ROIを記載

### 契約書のカスタマイズ

1. **契約書番号**
   - `API-XXXX-YYYYMMDD` → 実際の契約書番号に置換

2. **社名・住所**
   - `[パートナー企業名]` → 実際の企業名
   - `[御社名]` → 実際の御社名
   - 住所、代表者名を記載

3. **料金プラン**
   - 別紙「料金表」を添付
