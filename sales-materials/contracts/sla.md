## 2. サービス稼働率

### 2.1 稼働率保証

当社は、以下の稼働率を保証します。

| プラン | 月間稼働率 | ダウンタイム上限（月間） |
|--------|-----------|---------------------|
| **Starter** | 99.0%以上 | 7時間12分以下 |
| **Business** | 99.5%以上 | 3時間36分以下 |
| **Enterprise** | 99.9%以上 | 43分12秒以下 |

### 2.2 稼働率の計算方法

```
稼働率（%）= (総時間 - ダウンタイム) ÷ 総時間 × 100
```

**総時間**: 1ヶ月間の時間（分単位）
**ダウンタイム**: HTTPステータスコード500または503が連続5分以上返される時間

### 2.3 ダウンタイムの除外

以下の事由によるダウンタイムは、稼働率の計算から除外します。

- 定期メンテナンス（事前通知あり）
- 緊急メンテナンス（セキュリティ対応等）
- お客様の環境に起因する障害
- 第三者のサービス（AWS、GCP等）の障害
- 天災、戦争、暴動等の不可抗力
- お客様の過失または本契約違反に起因する障害

## 4. サポート対応時間

### 4.1 サポート窓口

| サポートチャネル | Starter | Business | Enterprise |
|----------------|---------|----------|-----------|
| **メール** | ✓ | ✓ | ✓ |
| **チャット** | - | ✓ | ✓ |
| **電話** | - | - | ✓ |

### 4.2 対応時間

**営業時間**: 平日10:00-18:00（日本時間、祝日除く）

| プラン | 初回応答時間 | 解決時間 |
|--------|-------------|---------|
| **Starter** | 48時間以内 | 5営業日以内 |
| **Business** | 24時間以内 | 3営業日以内 |
| **Enterprise** | 4時間以内 | 1営業日以内 |

### 4.3 優先度別対応

#### Severity 1（緊急）
- 定義: 本サービスが完全に利用不可
- 初回応答: 4時間以内（Enterpriseのみ）
- 対応: 24時間体制で復旧作業

#### Severity 2（高）
- 定義: 主要機能が利用不可
- 初回応答: 8時間以内（Business以上）
- 対応: 営業時間内に復旧作業

#### Severity 3（中）
- 定義: 一部機能に支障
- 初回応答: 24時間以内（Business以上）
- 対応: 次回メンテナンスで対応

#### Severity 4（低）
- 定義: 軽微な不具合・質問
- 初回応答: 48時間以内（全プラン）
- 対応: 順次対応

### 5.1 定期メンテナンス

**頻度**: 月1回

## 6. 障害対応

### 6.1 障害検知

当社は、以下の方法で24時間365日、本サービスを監視します。

- APIエンドポイントの死活監視（1分間隔）
- 応答時間の監視（1分間隔）
- エラー率の監視（1分間隔）
- リソース使用率の監視（5分間隔）

### 6.2 障害通知

**通知方法**:
- メール（全お客様）
- 管理画面（ステータスページ）
- SMS（Enterpriseのみ、希望者）

**通知内容**:
- 障害発生時刻
- 影響範囲
- 復旧見込み時刻
- 回避策（可能な場合）

### 6.3 復旧プロセス

```
障害検知
  ↓
初動対応（15分以内）
  ↓
原因調査（30分以内）
  ↓
復旧作業
  ↓
動作確認
  ↓
復旧完了通知
  ↓
事後報告書作成（24時間以内）
```

### 6.4 事後報告書

障害発生後24時間以内に、以下を含む報告書を提供します。

- 障害発生日時
- 障害内容
- 影響範囲
- 原因
- 対応内容
- 再発防止策

## 8. 免責事項

### 8.1 SLA適用除外

以下の事由による障害は、SLAの適用対象外とします。

1. **お客様の環境に起因する問題**
   - ネットワーク障害
   - ファイアウォール設定
   - DNS設定
   - ブラウザ設定

2. **第三者のサービス障害**
   - AWS、GCP、Azure等のクラウドサービス
   - CDN（Content Delivery Network）
   - DNS（Domain Name System）

3. **不可抗力**
   - 天災（地震、台風、洪水等）
   - 戦争、暴動、テロ
   - 伝染病、パンデミック
   - 政府機関の命令

4. **お客様の行為に起因する問題**
   - 過度なAPI呼び出し
   - 不正アクセス
   - 本契約違反

5. **定期メンテナンス**
   - 事前通知済みのメンテナンス
