## 📊 自動計算エリア

### 2. 現状コスト試算

#### 外注費
```
外注費 = 月間動画本数 × 外注比率 × 外注単価
```

**計算例**（月20本、外注比率80%、単価5万円の場合）:
```
外注費 = 20本 × 0.8 × ¥50,000 = ¥800,000/月
```

#### 内製工数コスト
```
内製コスト = 月間動画本数 × 内製比率 × 制作工数 × 人件費単価
```

**計算例**（月20本、内製比率20%、工数20時間/本、人件費5,000円/時間の場合）:
```
内製コスト = 20本 × 0.2 × 20h × ¥5,000 = ¥400,000/月
```

#### ディレクション工数
```
ディレクション工数 = 月間動画本数 × 平均工数（3時間/本） × 人件費単価
```

**計算例**（月20本、人件費5,000円/時間の場合）:
```
ディレクション工数 = 20本 × 3h × ¥5,000 = ¥300,000/月
```

#### 素材購入費
```
素材購入費 = ¥[ ] 円/月
```

#### 月間総コスト
```
月間総コスト = 外注費 + 内製コスト + ディレクション工数 + 素材購入費
```

**計算例**:
```
月間総コスト = ¥800,000 + ¥400,000 + ¥300,000 + ¥100,000 = ¥1,600,000/月
```

#### 年間総コスト
```
年間総コスト = 月間総コスト × 12ヶ月
```

**計算例**:
```
年間総コスト = ¥1,600,000 × 12 = ¥19,200,000/年
```

### 4. コスト削減効果

#### 月間削減額
```
月間削減額 = 現状月間総コスト - 導入後月間総コスト
```

**計算例**:
```
月間削減額 = ¥1,600,000 - ¥200,000 = ¥1,400,000/月
```

#### 削減率
```
削減率（%）= (月間削減額 ÷ 現状月間総コスト) × 100
```

**計算例**:
```
削減率 = (¥1,400,000 ÷ ¥1,600,000) × 100 = 87.5%
```

#### 年間削減額
```
年間削減額 = 月間削減額 × 12ヶ月
```

**計算例**:
```
年間削減額 = ¥1,400,000 × 12 = ¥16,800,000/年
```

#### 3年間累計削減額
```
3年間累計削減額 = 年間削減額 × 3年
```

**計算例**:
```
3年間累計削減額 = ¥16,800,000 × 3 = ¥50,400,000
```

### 5. 初期投資とROI

#### 初期投資
```
初期投資 = 導入設定費用 + 初期トレーニング + システム統合費用
```

**標準パッケージ**:
```
初期投資 = ¥100,000 + ¥50,000 + ¥0 = ¥150,000
```

**システム統合あり**:
```
初期投資 = ¥100,000 + ¥50,000 + ¥200,000 = ¥350,000
```

#### 投資回収期間
```
投資回収期間（月）= 初期投資 ÷ 月間削減額
```

**計算例**（初期投資15万円、月間削減額140万円の場合）:
```
投資回収期間 = ¥150,000 ÷ ¥1,400,000 = 0.11ヶ月（約3日）
```

#### 年間ROI
```
年間ROI（%）= ((年間削減額 - 初期投資) ÷ 初期投資) × 100
```

**計算例**:
```
年間ROI = ((¥16,800,000 - ¥150,000) ÷ ¥150,000) × 100 = 11,100%
```

#### 3年間ROI
```
3年間ROI（%）= ((3年間累計削減額 - 初期投資) ÷ 初期投資) × 100
```

**計算例**:
```
3年間ROI = ((¥50,400,000 - ¥150,000) ÷ ¥150,000) × 100 = 33,500%
```

## 付録: ROI計算式一覧

### 基本計算式

**現状月間総コスト**:
```
= 外注費 + 内製コスト + ディレクション工数 + 素材購入費
```

**外注費**:
```
= 月間動画本数 × 外注比率 × 外注単価
```

**内製コスト**:
```
= 月間動画本数 × 内製比率 × 制作工数 × 人件費単価
```

**ディレクション工数**:
```
= 月間動画本数 × 平均工数（3h/本） × 人件費単価
```

**導入後月間総コスト**:
```
= API利用料 + 運用工数 + 素材購入費
```

**運用工数**:
```
= 月間動画本数 × 運用時間（1h/本） × 人件費単価
```

**月間削減額**:
```
= 現状月間総コスト - 導入後月間総コスト
```

**削減率**:
```
= (月間削減額 ÷ 現状月間総コスト) × 100
```

**投資回収期間**:
```
= 初期投資 ÷ 月間削減額
```

**年間ROI**:
```
= ((年間削減額 - 初期投資) ÷ 初期投資) × 100
```
