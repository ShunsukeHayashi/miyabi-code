# A2A Broadcast

全エージェントにメッセージをブロードキャストします。

## 使用方法

/a2a-broadcast [メッセージ]

## 実行内容

P0.2プロトコルに従い、全エージェントペイン(%18-%23)にメッセージを送信:

```bash
source ./a2a.sh && a2a_broadcast "$ARGUMENTS"
```

## メッセージフォーマット

```
[System] ブロードキャスト: <メッセージ内容>
```

ユーザーが指定したメッセージを全エージェントに配信してください。
引数がない場合は、ブロードキャストしたいメッセージを確認してください。
