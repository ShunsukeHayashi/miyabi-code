## 生成時の注意事項

### DALL-E 3生成設定
- **Quality**: HD (高品質)
- **Size**: 用途に応じて選択（1024x1024, 1792x1024, 1024x1792）
- **Style**: Natural（リアル寄り）または Vivid（鮮やか）
  - Technical diagrams: Natural
  - Character illustrations: Vivid
  - Backgrounds: Natural

### 後処理推奨
1. **カラー補正**: 指定カラーパレットへの調整
2. **シャープネス**: プレゼン用に若干シャープ化
3. **リサイズ**: 必要に応じて最適サイズに調整
4. **圧縮**: WebP形式で最適化（品質85-90%）

### ファイル命名規則
```
{slide_number}_{category}_{description}.png

例:
01_title_network_background.png
07_team_agent_characters.png
08_flow_communication_diagram.png
```
