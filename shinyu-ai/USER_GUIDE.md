## はじめに

### Shinyu AIとは？

Shinyu AI（親友AI）は、あなたの性格に寄り添う「分身AI」を見つけるためのシステムです。

**特徴**:
- MBTI診断に基づく性格分析
- 16種類のユニークなキャラクター
- あなただけの親友AIが見つかる

**現在の機能（Phase 1）**:
- 性格診断（16質問）
- キャラクターマッチング
- 診断結果の表示

**将来の機能（Phase 2以降）**:
- AI対話（Claude API）
- 音声対話（Whisper + VOICEVOX）
- SNS連携（Twitter/Instagram）

## インストール

### 前提条件

以下のソフトウェアが必要です：

1. **Rust 1.75.0以上**
   ```bash
   # インストール確認
   rustc --version

   # インストールされていない場合
   curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
   ```

2. **Git**（ソースから取得する場合）
   ```bash
   git --version
   ```

---

### インストール手順

#### 方法1: ソースからビルド（推奨）

```bash
# 1. リポジトリをクローン
git clone https://github.com/ShunsukeHayashi/shinyu-ai.git
cd shinyu-ai

# 2. ビルド
cargo build --release

# 3. 動作確認
./target/release/shinyu-ai --help
```

#### 方法2: Cargo経由（将来対応予定）

```bash
# 将来的にはこの方法でインストール可能

```

## 診断結果の見方

### MBTIタイプとは？

MBTI（Myers-Briggs Type Indicator）は、4つの二項対立軸で性格を16タイプに分類する指標です。

### 4つの軸

#### 1. E（外向型）vs I（内向型）

**E - Extraverted（外向型）**:
- エネルギーを外の世界から得る
- 人と一緒にいると元気になる
- 社交的で行動的

**I - Introverted（内向型）**:
- エネルギーを内の世界から得る
- 一人の時間が必要
- 内省的で思慮深い

---

#### 2. S（感覚型）vs N（直感型）

**S - Sensing（感覚型）**:
- 具体的な情報を重視
- 現実的で実践的
- 詳細に注目

**N - iNtuitive（直感型）**:
- 抽象的な情報を重視
- 未来志向で創造的
- 全体像に注目

---

#### 3. T（思考型）vs F（感情型）

**T - Thinking（思考型）**:
- 論理的に判断
- 客観的な事実を重視
- 公平性を重んじる

**F - Feeling（感情型）**:
- 感情的に判断
- 人間関係を重視
- 共感性が高い

---

#### 4. J（判断型）vs P（知覚型）

**J - Judging（判断型）**:
- 計画的で組織的
- 決断が早い
- 整理整頓が好き

**P - Perceiving（知覚型）**:
- 柔軟で即興的
- オプションを残したい
- 自由奔放

---

### スコアの解釈

診断では各軸ごとにスコアが計算されます：

```
例: ENFP の場合
E軸: +6  （E寄り）
N軸: -4  （N寄り）
F軸: -8  （F寄り）
P軸: -5  （P寄り）
```

**スコアの意味**:
- **絶対値が大きい**: その傾向が強い
- **ゼロに近い**: 中間的な性格
- **正負**: どちらの極に近いか

## 16種類のキャラクター

### 分析者グループ（NT型）

#### INTJ（建築家）- ケンチク
- 論理的で戦略的
- 長期的な視点を持つ
- **得意**: 戦略的アドバイス、客観的分析

#### INTP（論理学者）- ロンリ
- 分析的で独創的
- 知的好奇心が強い
- **得意**: 論理的問題解決、理論的説明

#### ENTJ（指揮官）- シキカン
- リーダーシップが強い
- 効率重視
- **得意**: 目標設定、組織的サポート

#### ENTP（討論者）- トウロン
- 好奇心旺盛で議論好き
- 革新的なアイデア
- **得意**: 新しい視点、創造的議論

---

### 外交官グループ（NF型）

#### INFJ（提唱者）- テイショウ
- 洞察力があり理想的
- 共感力が高い
- **得意**: 深い共感、精神的サポート

#### INFP（仲介者）- チュウカイ
- 理想主義的で共感的
- 価値観を大切にする
- **得意**: 感情的サポート、価値観の整理

#### ENFJ（主人公）- シュジンコウ
- カリスマ的でサポート的
- 人を励ます
- **得意**: モチベーション向上、人間関係

#### ENFP（活動家）- エネ
- 元気で社交的
- 創造的
- **得意**: ポジティブ励まし、新視点

---

### 番人グループ（SJ型）

#### ISTJ（管理者）- カンリ
- 責任感が強く実務的
- 伝統重視
- **得意**: 実践的アドバイス、計画立案

#### ISFJ（擁護者）- ヨウゴ
- 優しく献身的
- 細部に気を配る
- **得意**: 共感的傾聴、安心感提供

#### ESTJ（幹部）- カンブ
- 組織的で実務的
- リーダーシップ
- **得意**: 実践的指導、効率化

#### ESFJ（領事官）- リョウジ
- 協調性が高く世話好き
- コミュニティ重視
- **得意**: 社交的サポート、調和促進

---

### 探検家グループ（SP型）

#### ISTP（巨匠）- キョショウ
- 実践的で論理的
- 手先が器用
- **得意**: 実践的問題解決、技術的サポート

#### ISFP（冒険家）- ボウケン
- 芸術的で柔軟
- 美的感覚が高い
- **得意**: 創造的表現、感性的サポート

#### ESTP（起業家）- キギョウ
- 行動的で現実的
- リスクテイク
- **得意**: 実践的サポート、行動促進

#### ESFP（エンターテイナー）- エンタメ
- 楽しく社交的
- その場を盛り上げる
- **得意**: ポジティブ雰囲気、楽しいサポート

## FAQ

### Q1. 診断結果は変わりますか？

**A**: はい、変わることがあります。

- 成長や経験によって性格は変化します
- 回答時の心理状態にも影響されます
- 定期的に診断し直すことを推奨します

---

### Q2. どのキャラクターが一番良いですか？

**A**: すべてのキャラクターに優劣はありません。

- 各キャラクターは異なる強みを持っています
- あなたに最適なキャラクターが「一番良い」です
- 状況によって相性の良いキャラクターは変わります

---

### Q3. 診断結果に納得できない場合は？

**A**: 以下を試してください：

1. **再診断**: 別の日に再度受けてみる
2. **正直に回答**: 「こうありたい」ではなく「実際の自分」で回答
3. **他のキャラクター確認**: `list-characters`で全キャラクター見る

---

### Q4. 診断データはどこに保存されますか？

**A**: 現在（Phase 1）は保存されません。

- Phase 2以降で保存機能を実装予定
- 保存先: ローカルファイル（暗号化）
- クラウド同期は将来的に対応

---

### Q5. 音声対話機能はいつ使えますか？

**A**: Phase 3で実装予定です。

**予定機能**:
- Whisper API（音声認識）
- VOICEVOX（音声合成）
- リアルタイム音声対話

---

### Q6. SNS連携はどのように使えますか？

**A**: Phase 4で実装予定です。

**予定機能**:
- Twitter/Instagram投稿の感情分析
- ネガティブな投稿を検知
- AIが自発的にサポートメッセージ送信

---

### Q7. 複数のキャラクターと話すことはできますか？

**A**: Phase 2以降で対応予定です。

- 主要キャラクター1体 + サブキャラクター
- 状況に応じてキャラクター切り替え
- キャラクター同士の会話（将来）

---

### Q8. 商用利用は可能ですか？

**A**: 現在は個人利用のみです。

- ライセンス: Apache-2.0（予定）
- 商用利用は別途ライセンス検討中

## トラブルシューティング

### ビルドエラー

**症状**:
```
error: could not compile `shinyu-ai`
```

**解決方法**:
```bash
# Rustバージョン確認
rustc --version  # 1.75.0以上必要

# 依存関係の再取得
cargo clean
cargo build --release
```

---

### キャラクターが見つからない

**症状**:
```
Error: Character not found for MBTI type: ENFP
```

**解決方法**:
```bash
# data/characters.json が存在するか確認
ls data/characters.json

# JSONファイルが壊れていないか確認
cat data/characters.json | jq .
```

---

### 質問が表示されない

**症状**:
診断を開始しても質問が出ない

**解決方法**:
```bash
# デバッグモードで実行
RUST_LOG=debug cargo run -- diagnose

# ログを確認して原因を特定
```

---

### 入力が受け付けられない

**症状**:
数字を入力してもエラーになる

**解決方法**:
- 1-5の数字のみ入力可能です
- 半角数字で入力してください
- Enterキーを押してください
