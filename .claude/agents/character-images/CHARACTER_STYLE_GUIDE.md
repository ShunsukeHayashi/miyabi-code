# キャラクタースタイルガイド - 一貫性確保のための設定

## 🎨 共通スタイル設定
- **アートスタイル**: 高品質アニメイラスト、京都アニメーション風
- **カラーパレット**: 各キャラクターに固有の色を設定
- **基本構成**: 全身像、特徴的な装飾品、背景要素

---

## 📋 各キャラクターの固定要素

### 1. しきるん (Shikiroon) - Coordinator Agent
- **髪色**: エメラルドグリーン、ショートヘア
- **目の色**: 明るい緑
- **服装**: 深緑の指揮者風ユニフォーム、金の飾緒
- **アクセサリー**: シャープな知的メガネ
- **持ち物**: 光るシアンの指揮棒
- **性格表現**: 権威的だが可愛らしい

### 2. つくるん (Tsukuroon) - CodeGen Agent  
- **髪色**: ダークネイビーブルー、乱れた髪
- **目の色**: 深い青
- **服装**: 特大の紺色パーカー（光る二進法パターン）
- **アクセサリー**: ゲーミングヘッドセット
- **持ち物**: ホログラフィックキーボード
- **性格表現**: 集中力が高く、やや疲れた様子

### 3. めだまん (Medaman) - Review Agent
- **髪色**: アイスブルー、顎までのボブカット
- **目の色**: シアン
- **服装**: 白のブレザー、水色の発光アクセント
- **アクセサリー**: リムレススマートグラス（データ表示）
- **持ち物**: ホログラフィックタブレット、スタイラス
- **性格表現**: 厳格でプロフェッショナル

### 4. みつけるん (Mitsukeroon) - Issue Agent
- **髪色**: 明るい黄色、スパイキーヘア
- **目の色**: アンバー
- **服装**: 黄色と黒のストリートウェアパーカー
- **アクセサリー**: 大型ヘッドホン
- **持ち物**: 未来的な虫眼鏡
- **性格表現**: エネルギッシュで行動的

### 5. まとめるん (Matomeroon) - PR Agent
- **髪色**: ラベンダーパープル、小さなポニーテール
- **目の色**: バイオレット
- **服装**: 紫と白の配達員風ジャケット
- **アクセサリー**: git-mergeアイコンのピン
- **持ち物**: 設計図入りメッセンジャーバッグ
- **性格表現**: 優しく親切

## 🎭 バリエーションタイプ

### A. 表情バリエーション
1. **通常** - デフォルトの表情
2. **集中** - 真剣な作業中
3. **成功** - タスク完了時の喜び
4. **困惑** - 問題発見時
5. **決意** - 解決に向かう時

### B. ポーズバリエーション
1. **スタンディング** - 基本立ちポーズ
2. **アクション** - 仕事中の動的ポーズ
3. **シッティング** - 座って作業
4. **ポインティング** - 指示・説明
5. **ビクトリー** - 成功のポーズ

### C. シチュエーションバリエーション
1. **オフィス** - 通常の作業環境
2. **サーバールーム** - 技術的環境
3. **会議室** - チームワーク場面
4. **屋外** - リラックスシーン
5. **バーチャル空間** - デジタル環境

## 🔧 生成プロンプトテンプレート

```
[キャラクター名] in [バリエーションタイプ], 
[固定要素をすべて含む],
[特定の表情/ポーズ/シチュエーション],
masterpiece quality anime illustration,
consistent character design,
[背景詳細]
```

## 📐 技術仕様
- **解像度**: 1024x1024 (1K) / 2048x2048 (2K)
- **アスペクト比**: 1:1
- **カラーモード**: RGB
- **ファイル形式**: PNG
