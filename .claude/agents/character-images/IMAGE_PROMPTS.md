## 使用方法

1. お好みのAI画像生成ツールを開く
2. 上記のプロンプトをコピー＆ペースト
3. 必要に応じて設定を調整:
   - アスペクト比: 1:1 または 3:4（ポートレート）
   - 解像度: 1024x1024以上推奨
   - スタイル: アニメ/マンガイラスト

4. 生成された画像を以下の名前で保存:
   - coordinator.png
   - codegen.png
   - honoka.png
   など
