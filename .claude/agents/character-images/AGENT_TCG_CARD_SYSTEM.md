## 🔥 属性システム

### 主属性（2種類）
1. **🖥️ Coding（コーディング）** - 技術系エージェント
2. **💼 Business（ビジネス）** - ビジネス系エージェント

### サブ属性（6種類）
- **⚡ Lightning（雷）** - 高速処理、即座の実行
- **🔥 Fire（炎）** - 攻撃的、破壊的変更
- **💧 Water（水）** - 適応性、流動的
- **🌱 Nature（自然）** - 成長、持続可能
- **🌙 Dark（闇）** - 分析、深層探索
- **✨ Light（光）** - 創造、革新

### 属性相性
```
⚡ → 💧 → 🔥 → 🌱 → ⚡
✨ ↔ 🌙
```

## 🎯 スキルシステム

### スキル種別
1. **アクティブスキル** - 能動的に発動
2. **パッシブスキル** - 常時発動
3. **シナジースキル** - 特定の組み合わせで発動
4. **究極スキル** - レベルMAX時に解放

### スキル例

#### しきるん（Coordinator）
- **パラレル・オーケストラ**: 複数エージェントの同時実行
- **タスクハーモニー**: チーム全体の効率+50%
- **究極・無限並列**: 制限なしの並列実行

#### つくるん（CodeGen）
- **コードバースト**: 瞬間的に大量コード生成
- **バグシールド**: 自動エラー修正
- **究極・創世記**: 完全自律コード生成

## 📈 成長・進化システム

### レベルシステム
- **初期レベル**: 1
- **最大レベル**: 100
- **進化可能レベル**: 30, 50, 70

### 進化段階
```
基本形 → 進化形 → 最終進化形 → 究極進化形
Lv.1    Lv.30    Lv.50       Lv.70

例: しきるん → しきるんEX → しきるんMASTER → しきるんΩ
```

### 経験値獲得方法
1. **タスク完了**: 10-100 EXP
2. **Issue解決**: 50-500 EXP
3. **PR作成**: 100-1000 EXP
4. **デプロイ成功**: 500-5000 EXP
5. **ユーザー評価**: ボーナスEXP

## 🎮 ゲームメカニクス

### バトルシステム
```
1. 速度順に行動
2. 属性相性によるダメージ補正
3. スキル発動によるコンボ
4. チームシナジーボーナス
```

### チーム編成
- **最大**: 5体
- **コスト制限**: レア度によるコスト
- **シナジー**: 特定の組み合わせでボーナス

### 特殊ルール
- **タイプ一致ボーナス**: 同タイプ3体以上で+20%
- **属性チェイン**: 相性良い属性の連携で追加ダメージ
- **進化ラッシュ**: バトル中の進化で全回復
