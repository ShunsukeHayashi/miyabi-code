## 3. オープニングパターン

### 高エンゲージメント型

```
【逆張り・煽り型】(最強)
- 「えー、まだ〇〇してるの？」
- 「〇〇信者、目を覚ませ。」
- 「まだ〇〇「〇〇」んの？」

【自虐・告白型】(高い)
- 「恥ずかしながら、私は〇〇」
- 「正直に言います。」
- 「実は〇〇なんです。」

【インパクト数字型】(効果的)
- 「神は７日で世界を作った。〇〇は〇時間で〇〇した。」
- 「5,000人に教えて気づいた〇〇」
- 「〇〇が〇倍速くなった」
```

## 4. 構成パターン

### パターンA: 逆張り + 根拠

```
[煽りオープニング]
↓
[問題提起 2-3行]
↓
[具体的なコード/例]
↓
[結論・教訓]
```

### パターンB: 数字 + 証拠

```
[衝撃の数字]
↓
[背景説明]
↓
[具体的なプロンプト/コード]
↓
[行動喚起]
```

### パターンC: 告白 + 学び

```
[自虐・告白オープニング]
↓
[具体的なエピソード]
↓
[そこから得た学び]
↓
[読者への示唆]
```
