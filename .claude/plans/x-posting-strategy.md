```

- 投稿案の自動生成
- エンゲージメント予測
- 最適投稿時間提案

- 実際の投稿実行
- PRO tier で利用可能

- 定期的なアカウント分析
- 競合分析
```

### 9.2 自動化フロー

```
1. トレンド検索 → WebSearch
2. 投稿案生成 → tsubuyakun_generate_x_post
3. 確認・編集 → 手動
4. 投稿実行 → tsubuyakun_post_to_x
```
