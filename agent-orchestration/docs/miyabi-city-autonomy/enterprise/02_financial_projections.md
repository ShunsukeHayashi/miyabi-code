## 4. 資金調達計画と返済スケジュール
- エクイティ: 30億（Year0）、想定希薄化を抑え自治性を確保。優先株/参加型劣後を検討。
- デット: つなぎ借入2億（Year3）、長期借入10億（Year0-1）。返済はYear5-10で均等1億/年＋利息。
- 補助金/助成: 年0-4で累計5億を想定（エネルギー、地域活性、観光、教育）。
- キャッシュマネジメント: 運転資金枠3億（コミットライン）を確保し、期末現金を年運営費1ヶ月以上維持。
