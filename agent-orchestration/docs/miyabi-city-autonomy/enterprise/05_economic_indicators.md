## 2. モニタリング運用
- ダッシュボード: 売上/CF/稼働率/自給率/補助依存度を月次更新。公共クラウド＋バックアップ。
- ゲート審査: 年1/3/5/7/10で「自治拡張・資金調達・CAPEX追加」を意思決定。KPI未達時は投資凍結とコスト縮減を自動トリガー。
- 第三者評価: ESG/コンプライアンス/住民満足度を外部機関で年次評価。

## 3. リスク早期警戒（早期警戒ライン）
- キャッシュ残高<運営費3ヶ月 → コスト削減/CF改善パッケージを即発動。
- 稼働率<60%が2期連続 → マーケ/価格改定、短期キャンペーン投入。
- エネルギー自給率<目標-10% → CAPEX前倒しor需給調整契約を強化。
- 住民参加率<50% → 合意形成プロセス再設計、情報開示強化。
