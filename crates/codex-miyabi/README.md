# codex-miyabi

**Status**: Internal
**Category**: CLI Tool

## Overview

Thin CLI to exercise miyabi-integration for Codex × Miyabi Phase 1. Provides a command-line interface for integrating OpenAI Codex with the Miyabi autonomous agent platform.

## Features

- Integration with miyabi-integration crate
- CLI interface for Codex operations
- Phase 1 functionality for Codex × Miyabi project

## Usage

```bash
codex-miyabi --help
```

## Dependencies

- `miyabi-integration` - Core integration module
- `clap` - Command-line argument parsing
- `tokio` - Async runtime
